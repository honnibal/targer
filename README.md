# TARGER: Neural Argument Mining at Your Fingertips

This page contains code of the Web application and web service based on the neural [argument tagger](http://github.com/achernodub/bilstm-cnn-crf-tagger). This repository shares code and data related to the following paper:

Artem Chernodub, Oleksiy Oliynyk, Philipp Heidenreich, Alexander Bondarenko, Matthias Hagen, Chris Biemann, and Alexander Panchenko (2019): [TARGER: Neural Argument Mining at Your Fingertips](https://www.inf.uni-hamburg.de/en/inst/ab/lt/publications/2019-chernodubetal-acl19demo-targer.pdf). In Proceedings of the 57th Annual Meeting of the Association of Computational Linguistics (ACL'2019). 

If you use the demo or would like to refer to it, please cite the paper mentioned above. 


## Quick run web application with docker
`docker-compose up`
Access the frontend at http://localhost:6001/ and the backend REST API at http://localhost:6000.
